# 🔌 IHLTVClient

**命名空间:** `SwiftlyS2.Shared.Engine`

**类型:** `interface`

**继承:** `IServerSideClient`

## 📝 属性

| 名称 | 类型 | 访问方法 | 描述 |
|------|------|--------|------|
| `Password` | `string` | get, set | 获取或设置 HLTV 客户端的密码，该密码用于在连接到 HLTV 服务器时验证客户端身份。此设置通常对希望访问 HLTV 服务器上特定功能或查看特定内容的客户端是必需的，它有助于确保只有经过授权的客户端才能访问敏感信息或在服务器上执行某些操作。 |
| `ChatGroup` | `string` | get, set | 获取或设置 HLTV 客户端的聊天组，用于确定该客户端将接收哪些聊天消息。这使得 HLTV 客户端能够根据特定组（如队伍聊天、全局聊天或观战聊天）过滤聊天消息，确保客户端仅接收相关信息，并专注于对其观看体验最为重要的内容。 |
| `LastSendTime` | `double` | get, set | 获取或设置 HLTV 客户端上次向服务器发送消息的时间，该时间用于跟踪客户端活动并管理客户端与服务器之间的通信。此信息可用于多种目的，例如限制消息速率以防止垃圾邮件、跟踪客户端参与度，或实现依赖于客户端与服务器交互时机的功能。 |
| `LastChatTime` | `double` | get, set | 获取或设置 HLTV 客户端最后一次发送聊天消息的时间，该信息用于跟踪客户端的聊天活动并管理客户端与服务器之间的通信。此信息可用于多种用途，例如对聊天消息实施速率限制以防止垃圾信息、跟踪客户端在聊天中的参与度，或实现依赖于客户端与服务端聊天系统交互时序的功能。 |
| `LastSendTick` | `int` | get, set | 获取或设置 HLTV 客户端最后一次向服务器发送消息的 tick，该值用于追踪客户端活动及管理客户端与服务器之间的通信。此信息可用于多种用途，例如对消息进行速率限制以防止垃圾信息、追踪客户端参与度，或实现依赖于基于游戏 tick 的客户端与服务端交互时序的功能。 |
| `FullFrameTime` | `int` | get, set | 获取或设置 HLTV 客户端的完整帧时间，该值用于追踪客户端性能并管理客户端与服务器之间的通信。此信息可用于多种目的，例如优化性能、追踪客户端参与度，或实现依赖基于帧时间的客户端与服务器交互时序的功能。 |
| `NoChat` | `bool` | get, set | 获取或设置一个值，指示 HLTV 客户端是否已禁用聊天功能；该功能可用于防止客户端接收或发送聊天消息。此功能通常用于主要专注于观看游戏的客户端，以避免因聊天消息而分散注意力；也可用于希望在仍能访问 HLTV 服务器其他功能的同时，避免聊天系统中潜在垃圾信息或滥用行为的客户端。 |
