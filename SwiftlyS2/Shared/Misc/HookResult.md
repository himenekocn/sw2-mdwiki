# 📋 HookResult

钩子的返回结果。

**命名空间:** `SwiftlyS2.Shared.Misc`

**类型:** `enum`

**继承:** `uint`

## 📋 枚举值

| 名称 | 值 | 描述 |
|------|------|------|
| `Continue` | `0` | 以下钩子及原始函数的执行将继续进行。 |
| `Stop` | `1` | 以下钩子和原始函数的执行都将被取消。此机制不适用于后置钩子。 |
| `Handled` | `2` | 以下钩子的执行将被取消，但原始函数将继续运行。此功能不适用于后置钩子（post hooks）。 |
