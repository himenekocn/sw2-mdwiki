# 📋 HookResult

钩子的结果。

**命名空间:** `SwiftlyS2.Shared.Misc`

**类型:** `enum`

**继承:** `uint`

## 📋 枚举值

| 名称 | 值 | 描述 |
|------|------|------|
| `Continue` | `0` | 以下钩子函数和原始函数的执行将继续。 |
| `Stop` | `1` | 以下钩子和原始函数的执行都将被取消。对后置钩子无效。 |
| `Handled` | `2` | 以下钩子的执行将被取消，但原始函数将继续执行。不适用于后置钩子。 |
| `CancelOriginal` | `3` | 以下钩子的执行将继续，原始函数将被取消。 |
