# 📦 Command

**命名空间:** `SwiftlyS2.Shared.Commands`

**类型:** `class`

**用法:** `Command`

## 🏷️ 属性用法

### 构造函数

```csharp
[Command("", false, "")]
```

**参数:**

- `name` (`string`)
- `registerRaw` (`bool`)
- `permission` (`string`)

```csharp
[Command("", false, "", "SwiftlyS2 registered command")]
```

**参数:**

- `name` (`string`)
- `registerRaw` (`bool`) = `false`
- `permission` (`string`) = `""`
- `helpText` (`string`) = `"SwiftlyS2 registered command"`
