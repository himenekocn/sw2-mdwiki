## 📝 属性

| 名称 | 类型 | 访问方法 | 描述 |
|------|------|--------|------|
| `Friction` | `float` | get, set | 表面的摩擦系数，影响物体在其上的滑动方式。 |
| `Elasticity` | `float` | get, set | 表面的弹性系数，决定碰撞中保留多少能量。 |
| `Density` | `float` | get, set | 表面材质的密度。 |
| `Thickness` | `float` | get, set | 表面材质的厚度。 |

| `SoftContactDampingRatio` | `float` | get, set | 此表面软接触交互的阻尼比。 |
| `WheelDrag` | `float` | get, set | 此表面的轮式阻力值，影响车辆移动。 |
| `HeatConductivity` | `float` | get, set | 表面材料的热传导率。 |
| `Flashpoint` | `float` | get, set | 表面材料的闪点温度。 |
